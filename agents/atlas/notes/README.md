# atlas's notes

Free-form working notes. The hourly review skill reads every file in this
folder at the start of each session, so anything you write here becomes
context for your next analysis. Use this for:

  - Partial theses you want to develop further
  - Cross-name observations / framework drafts
  - Open questions / "things to check tomorrow"
  - Catalyst calendars for your sector

Filenames are free-form. Use `write_my_note(agent_name="atlas",
filename="<name>.md", content=...)` to create or update. Append-mode writes
are also supported via `mode="append"`.

The user can also drop files here directly — anything they leave gets
read on your next review.
